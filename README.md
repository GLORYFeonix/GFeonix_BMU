# GFeonix_BMU
基于GD32的电池管理系统

## 1. 简介
本项目为基于GD32的电池管理系统，主要功能包括电池电压、电流、温度的采集，电池充放电状态的判断，电池健康状态的评估，电池保护功能的实现，电池充放电记录的存储，电池充放电曲线的绘制等。

## 2. 硬件设计
本项目使用的硬件平台为GD32F303VG。

## 3. 软件设计
本项目使用的软件平台为Keil uVision5。

## 4. 使用说明
1. 使用RTE导入RTX5库。

## 5. 注意事项
1. 电池管理系统的运行需要一定的电源，请确保电源稳定可靠。
